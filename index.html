Students Grades Web Service API.

This web application created to give details about Shenkar students.
All the requests should be in GET method.
The students details returned in JSON objects.

Show All Students Details:

https://targil1.herokuapp.com/getAllStudent

Show Grade By Specific ID:

https://targil1.herokuapp.com/getStudGradeById/xxx

Parameters:
xxx - the id of the specific student

Throws:
error - if ID does not exist in the database.

Header:
header getStudGradeById - if ID does not exist in the database.

Show students in range of grades:

https://targil1.herokuapp.com/betweengrade?from=80&to=90

Parameters:
from - Specify the beginning of the grade range
to - Specify the end of the grade range

Throws:
error - if there isn't grades in the range that you specified

Header:
header betweenGrade - if there isn't grades in the range that you specified
